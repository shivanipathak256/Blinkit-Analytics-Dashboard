📊 Blinkit Analytics Dashboard
📌 Project Overview

This project presents an interactive analytics dashboard built using Blinkit grocery sales data. The objective is to analyze sales performance, product distribution, and outlet trends to generate meaningful business insights.

Blinkit is a quick-commerce grocery delivery platform in India that offers instant delivery of daily essentials.

🎯 Objectives

- Analyze overall sales performance

- Identify top product categories

- Understand outlet performance by size and location

- Track customer rating and product visibility patterns

🛠 Tools & Technologies

- Power BI – Dashboard creation

- Excel – Data source and preprocessing

- Data Analysis & Visualization

📂 Dataset

The dataset contains grocery store sales information including:

- Item Type

- Fat Content

- Outlet Size

- Outlet Location Type

- Sales

- Rating

- Item Visibility

📊 Key KPIs

- Total Sales

- Average Sales

- Number of Items

- Average Customer Rating

📈 Dashboard Insights

- Fruits & Vegetables and Snack Foods are among the top-selling categories.

- Tier-3 locations generate the highest sales compared to other tiers.

- Medium-sized outlets contribute significantly to total revenue.

- Customer ratings remain around 3.9 out of 5, indicating moderate satisfaction.

🚀 Project Outcome

The dashboard helps transform raw retail data into actionable insights, enabling better understanding of customer preferences, product performance, and outlet effectiveness.
